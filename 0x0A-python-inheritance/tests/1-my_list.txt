Using ``my_list``
-------------------

Import file
>>> MyList = __import__('1-my_list').MyList

>>> lista = MyList()

>>> print(lista)
[2, 1]

>>> lista.append(-1)
>>> lista.print_sorted()
[-1, 1, 2]

>>> lista is aux
False
